本项目基于 Astro 框架构建，旨在打造一个开源的古诗词静态网站，专注于展示与分享中国悠久的古诗词。以“星河”为意象，希望将浩瀚如繁星的诗词瑰宝，以现代技术与古典韵味交织的方式呈现，为诗词爱好者打造一个优雅、美观、轻量且响应迅速的诗词平台，实现近乎毫秒级的访问体验。
